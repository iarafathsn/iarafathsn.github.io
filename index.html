SmartThings (iOS)

The SmartThings iOS application is Samsung’s IoT platform that enables users to connect, monitor, and control a wide range of smart devices from their iPhone or iPad. It supports various devices, including TVs, lights, security cameras, and air conditioners, allowing seamless automation and remote access. Users can create customized routines, such as turning off lights and locking doors at bedtime, or automate actions based on location and sensor triggers.

The app also features MapView, a recently developed addition that visually maps IoT devices for easier interaction. Energy monitoring helps track power consumption, while real-time notifications and alerts keep users informed about security events, motion detection, and emergency situations. Integration with Bixby, Google Assistant, and Alexa allows for voice control, and Apple Watch support ensures quick access.

Additionally, SmartThings Find helps locate lost Samsung devices, and multi-user access lets family members share control of smart home devices with assigned permissions. The app continuously evolves, introducing new features every quarter to enhance the user experience and expand smart home capabilities.

5D Viewer

5D Viewer is standalone software that can be installed in laptops/PCs with Microsoft Windows. This product lets users use their computers to review, analyze, edit, and measure the volume data exported from ultrasound equipment via storage media such as USB drive. Since this product reads 3D volume data, users can review the test results of patients more quickly and easily. This function allows them to check 3D image results without using an ultrasound system, helping them conduct more tests with the ultrasound system. 5D Viewer only supports the DICOM files containing the volume data created by Samsung Medison. In other words, you only use the volume data (.mvl, sty) in DICOM files.

5D Viewer is not compatible with the DICOM files from a third party. 5D Viewer uses the volume data of Samsung Medison's WS80A, RS80A, and HS70A Ultrasound systems. The Volume Data contains the raw data obtained during the scan as well as the information required for rendering (e.g., Geometry, View mode, etc.). The patient information is not relevant to the rendering, so it is not included in the Volume Data.
